# zig-gba

```
git clone https://github.com/zig-homebrew/zig-gba
cd zig-gba/
zig build run
```

## Dependencies

- [zig](https://ziglang.org/download/)
- [devkitPro](https://devkitpro.org/wiki/Getting_Started)
